
Everyday helper

Command Line for now but I want to add some NLP/Speech Recognition in there 

Current features :

- Date
- Weather
- Inspirationnal quote of the day
- News headline

Dependencies :

- forecastio (Wrapper for Dark Sky API)
- gTTS (to enable the device to speak)
- mpg123 (to speak)

APIs used :

- News API : https://newsapi.org/
- Dark Sky API : https://darksky.net/forecast/48.86,2.35/si24/fr
